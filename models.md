# FGO 改模教程

本教程介绍如何利用 **“Mesh 顶点坐标与 UV 二进制原位回填法”** 对 战斗模型进行 3D 模型形变和图集扩展修改。

> [!NOTE]  
> **为什么采用此方法？**
> 
> 传统的 Unity 场景重装与重打包流程极易丢失官方包中原本的骨骼绑定、碰撞体配置等隐式设置，导致模型在游戏内出现卡死或无法动弹等问题。(如果你有相关知识能解决这个问题那就可以完全不需要按照本工作流来, 可以自己修改骨骼添加模型测试)
> 
> 本方法采用**不重装引擎、不进行二次打包**的数据直写工作流。在**不增删顶点**的前提下，允许任意修改已有模型的 3D 形状与 UV 采样范围。

---

## 1. 环境与工具准备

在开始前，请下载并安装以下工具：

1. **UABEANext**：用于解包/回填游戏资源。  
   👉 [官方下载地址](https://nightly.link/nesrak1/UABEANext/workflows/build-windows/master/uabea-windows.zip)
2. **AssetStudio**：用于可视化预览并导出官方模型的 FBX 格式。  
   👉 [官方发布页](https://github.com/Razviar/assetstudio/releases)
3. **Blender**：用于修改 3D 形状和排列 UV 坐标。  
   👉 [官方下载地址](https://www.blender.org/download/)
4. **Photoshop 或任意平面绘图软件**：用于修改和扩展图集。

5. 相关附件下载 https://github.com/Jekyell/F_Replace/releases/tag/models

## 2. 修改流程详解

本教程以妖兰2破（304810）加装“发饰角”为例。由于不可增加顶点，我们将通过“把原本的刘海/发鬓拉长放大”来重塑为角，并在扩展后的图集区画上新发饰。

### 步骤一：提取官方资源

1. 打开 **AssetStudio**，将游戏包文件 `304810` 拖入其中。
2. 展开下列的文件树找到 `body_level_2`。
3. 点击上方菜单 **Model -> Export selected objects (merge)**，导出得到一个 **`.fbx`** 模型文件。
4. 打开 **UABEANext**，拖入同一个游戏文件 `304810`。
5. 在资源列表点击 Name 表头进行排序，找到 `body_level_2`。
6. 选中它，点击上方菜单的 **Export**，在导出窗口右下角选择 **`.txt`** 格式导出，重命名为 **`old_mesh.txt`**。
7. 在列表顶部找到 `304810`（类型为 Texture2D），选中它并点击上方的 **Plugins -> Export Texture2D/Sprite**，导出为图集。

---

### 步骤二：扩展图集

1. 用 Photoshop 打开刚导出的 `304810.png` 图集。
2. 由于原图集已经被挤满，所以把画布宽度扩展两倍。
3. 在右侧新扩展的空白区，绘制或贴上发饰图案。
4. 保存为 PNG 格式。

---

### 步骤三：Blender 3D 塑形与 UV 对齐

1. 打开 Blender，按 `A` 全选并按 `Delete` 清空场景。
2. 点击 **文件 -> 导入 -> FBX**，导入在步骤一中得到的 `.fbx` 文件。
3. 切换到顶部的 **UV 编辑** 标签页。
4. **加载图集**：在左侧 UV 编辑器顶部点击“打开”，载入你在步骤二中扩展的两倍宽 PNG 图集。
5. **缩放复原 UV**：由于图集被横向拉宽了一倍，原本的 UV 布局会被拉伸。在左侧 UV 区域按快捷键：
   * **`A`**（全选 UV）
   * **`S -> X -> 键盘输入 0.5 -> 回车`**（沿 X 轴缩小一倍，锁定在左半边）
   * **`G -> X -> 键盘输入 -0.25 -> 回车`**（沿 X 轴向左平移，使其对齐左侧的原图集区域）
6. **移动与塑形发饰**：
   * 在右侧 3D 视图中按 **`Tab`** 进入编辑模式。
   * 右上角点击“选项”下拉菜单，勾选 **校正面属性** 和 **保留已连接项**。
   * 左侧uv区域找到原发鬓的顶点，鼠标悬停在上方按 **`L` 键**，选中该发鬓的整片 UV/网格。
   * 按 **`G` 键** 将它的 UV 整体移动到右侧我们新绘制的发饰图案上
   * uv区域中点击其中一个顶点，接着在右边区域按 **`G` 键**拉伸模型，此时拉伸会同步到我们左边的uv网格
7. 调整完成后，点击 **文件 -> 导出 -> Wavefront (.obj)**：


---

### 步骤四：脚本一键数据缝合

1. 将 `old_mesh.txt`、`body_modified.obj` 和缝合脚本 `patch_mesh.py` 放置在同一个文件夹内。
2. 打开 CMD 终端，输入命令运行脚本：
   ```bash
   python patch_mesh.py
   ```
3. 脚本会自动进行 3D 空间姿态回归对齐，识别图集格式（Float16 / Float），锁定 Stream 步长。如果网格没有增删顶点，将自动采用 **1:1 无损顺序直接映射** 回写坐标和新 UV。
4. 运行完成后，目录中会生成已缝合好的网格文本 **`patched_mesh.txt`**。

---

### 步骤五：回填并打包游戏包体

1. 回到 **UABEANext**。
2. 选中 `body_level_2`（Mesh 资源），点击上方菜单的 **Import**，导入 **`patched_mesh.txt`**。
3. 选中图集 `304810`（Texture2D 资源），点击上方的 **Plugins -> Edit Texture**。
4. 在弹出的窗口里点击 **Load texture...**，选择你修改后的两倍宽 PNG 图集导入。
5. 点击 **File -> Save all** 保存。
6. 将修改后的 `304810` 文件放到 `/Mod/Models` 文件夹中。
7. 打开游戏测试

---

## 3. 注意事项

> [!WARNING]  
> **顶点数禁忌**  
> 在 Blender 里编辑模型时，**千万不要增加或删除任何面与顶点**（即不要使用细分网格、挤出新面、合并其他物体等）。本直写工作流严格要求新旧模型的顶点索引一一对应，否则回填会报错或在游戏里花屏。
